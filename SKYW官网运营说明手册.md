## 2. 后台登录与权限

### 登录方式

后台采用**密码直接登录**（单一管理员账号，无用户名概念）。

| 项目 | 内容 |
|------|------|
| 登录地址 | `/admin`（自动跳转到登录页） |
| 管理员密码 | `skyw2026admin` |
| 登录有效期 | 7 天（Token 过期后需重新登录） |

### 权限说明

当前后台为**单一超级管理员**模式，登录后可操作所有功能，无分级权限区分。未来如需添加多个运营账号，需联系开发人员扩展权限系统。

### 安全建议

- 请勿将密码分享给无关人员
- 如需更换密码，联系开发人员修改服务器环境变量 `ADMIN_PASSWORD`
- 退出登录请点击侧边栏底部「退出登录」

## 4. 文章管理

**地址**：`/admin/articles`

### 4.1 文章列表

文章列表展示以下字段：

| 字段 | 说明 |
|------|------|
| 标题 | 文章中文标题 |
| 分类 | 公司新闻 / 市场分析 / 行业洞察 |
| 日期 | 发布日期 |
| GEO 得分 | AI 搜索优化评分（0–100 分），绿色≥80，黄色≥50，红色<50 |
| 状态 | 已发布 / 草稿，以及是否「⭐ 推荐」置顶 |
| 操作 | 编辑 / 删除 |

**分类筛选**：可按「全部 / 公司新闻 / 市场分析 / 行业洞察」筛选文章。

### 4.2 发布新文章

点击右上角「发布新文章」或访问 `/admin/articles/new`。

**基本信息**（必填）：

| 字段 | 说明 |
|------|------|
| Slug（URL标识） | 英文小写，用横线连接，如 `skyw-2026-q1-report`，一旦发布不要修改（影响外链） |
| 分类 | 三选一 |
| 日期 | 格式 `YYYY-MM-DD` |

**封面图**：
- 推荐尺寸：**1200×630 px**，比例 16:9
- 格式：JPG / PNG / WebP，文件 ≤ 800 KB
- 输入图片的完整 URL（如 Unsplash、CDN 链接）
- 填写后可实时预览

**三语内容**（点击标签切换语言）：

| 语言 | 标签 |
|------|------|
| 简体中文 | 简中 |
| 繁体中文 | 繁中 |
| English | EN |

每种语言需填写：**标题** / **摘要（Excerpt）** / **正文（Markdown）**

> **摘要字数建议**：40–155 字，影响 Google 搜索结果的描述文字和 GEO 得分。

**SEO 增强字段**：

| 字段 | 说明 |
|------|------|
| 关键词 | 逗号分隔，如 `香港基金,投资,IPO`，影响搜索排名 |
| 首页推荐置顶 | 勾选后该文章会在首页「行业洞察」区域优先展示，并带⭐标记 |

**SEO 预览**（实时显示）：
- Google 搜索结果预览（标题/URL/描述）
- 社交分享卡片预览（OG Image）

**发布状态**：
- **已发布**：公开可见，被搜索引擎抓取
- **草稿**：仅后台可见，不对外公开

### 4.3 GEO 得分说明

GEO 得分反映文章对 AI 搜索引擎（ChatGPT、Perplexity 等）的友好程度：

| 项目 | 分值 |
|------|------|
| 标题长度 10–40 字 | +20 |
| 摘要长度 40–155 字 | +25 |
| 封面图已设置 | +20 |
| 关键词已填写 | +20 |
| 正文长度 ≥ 200 字 | +15 |

建议所有发布文章的 GEO 得分达到 **80 分以上**。

## 8. SEO / GEO 优化

**地址**：`/admin/seo`

这是专为 AI 搜索引擎优化（GEO）设计的配置页，分三个标签页：

### 标签一：AI 爬虫 & 公司信息

**GEO 就绪评分**（0–100 分）：
- 实时显示 6 项检查，帮助判断 AI 搜索优化程度
- 建议所有项目显示绿色 ✅

**AI 爬虫权限**：
- 控制哪些 AI 爬虫可以访问网站
- 建议全部开启（默认已全部开启）
- 设置后自动同步到 `/robots.txt`

| 爬虫 | 对应产品 |
|------|---------|
| GPTBot | ChatGPT / OpenAI |
| ClaudeBot | Claude / Anthropic |
| PerplexityBot | Perplexity AI 搜索 |
| Google-Extended | Google AI Overview / Gemini |
| YouBot | You.com |
| CCBot | 多个 AI 训练数据来源 |

**公司结构化数据**：

| 字段 | 建议内容 |
|------|---------|
| 公司 AI 简介 | 50–150 字，用简洁中文介绍公司核心业务和服务对象，供 AI 生成回答时引用 |
| 成立年份 | 如 `2018` |
| 行业描述 | 如 `香港投资基金管理，专注另类资产` |

> 这些信息会写入 Google 可识别的 JSON-LD 结构化数据，让 AI 在回答用户问题时更准确地引用公司信息。

### 标签二：FAQ 常见问题

添加公司常见问题与答案：
- 问题和答案会生成 **Google FAQPage 结构化数据**
- 有助于在 Google 搜索结果中显示富文本摘要
- AI 搜索引擎（Perplexity、ChatGPT）回答用户问题时会优先引用
- **建议至少添加 5 条**，覆盖：公司介绍、服务范围、投资门槛、联系方式等

### 标签三：AI 内容地图（llms.txt）

- 系统已自动生成 `/llms.txt` 文件，供 AI 了解网站结构
- 可在「自定义补充内容」框中添加额外说明，如特别声明、免责条款等
- 点击「预览 llms.txt」可查看当前生成内容

### 三个自动生成文件（无需手动维护）

| 文件 | 地址 | 作用 |
|------|------|------|
| robots.txt | `/robots.txt` | 告知爬虫访问权限，已配置 AI 爬虫 |
| sitemap.xml | `/sitemap.xml` | 网站所有页面的索引，提交给 Google |
| llms.txt | `/llms.txt` | AI 友好内容地图（新标准） |

## 10. 图片上传规范

> 网站所有图片均通过**填写外部图片 URL** 的方式设置（不支持直接上传文件），建议使用以下图床服务：

**推荐图床**：
- [Cloudinary](https://cloudinary.com)（免费，专业图片 CDN）
- [Unsplash](https://unsplash.com)（免费高质量图库）
- 公司自有 CDN / 对象存储（OSS、S3）

### 各场景图片规范

| 场景 | 推荐尺寸 | 比例 | 最大文件大小 |
|------|---------|------|------------|
| 文章封面图 | 1200 × 630 px | 16:9 | 800 KB |
| 业务板块背景图 | 1920 × 1080 px | 16:9（严格） | 1.5 MB |
| 网站 OG 分享图 | 1200 × 630 px | 16:9 | 800 KB |
| 办公室图片（如有） | 800 × 600 px | 4:3 | 500 KB |

### 使用 Unsplash 获取图片 URL

1. 访问 [unsplash.com](https://unsplash.com)
2. 搜索关键词（如 `Hong Kong finance`、`investment`）
3. 点击图片 → 右键「复制图片地址」
4. 或点击「下载」→ 复制下载链接

## 11. 发布流程建议

### 新文章发布标准流程

```
1. 准备内容
   ├── 确定标题（10–40 字）
   ├── 准备简体中文、繁体中文、英文三语版本
   ├── 准备封面图（1200×630，上传到图床获取 URL）
   └── 确定关键词（5–10 个，逗号分隔）

2. 进入后台发布
   ├── 访问 /admin/articles/new
   ├── 填写 Slug（英文，用横线连接）
   ├── 选择分类（公司新闻/市场分析/行业洞察）
   ├── 填写封面图 URL
   ├── 依次填写三语：标题 + 摘要（40–155字）+ 正文
   ├── 填写关键词
   └── 确认 GEO 得分 ≥ 80 分

3. 发布设置
   ├── 重要或当前主推文章：勾选「首页推荐置顶」
   └── 确认无误后将状态改为「已发布」

4. 验收
   └── 访问 https://skyw.group/zh-CN/insights 查看效果
```

### 更新网站设置流程

1. 修改内容后点击「保存」
2. 等待 2-3 秒，出现「已保存」提示即成功
3. 刷新前台页面验证效果

## 12. 常见问题 FAQ

**Q：修改文章后前台没有立即更新？**
A：网站部分页面使用了 CDN 缓存，正常情况下 1–5 分钟内生效。如超过 10 分钟仍未更新，可在 Netlify 管理面板清除缓存，或联系开发人员。

**Q：图片填写了 URL 但预览显示不出来？**
A：请检查：①URL 是否完整（以 https:// 开头）；②图片服务器是否允许跨域访问（CORS）；③Unsplash 需使用直链，不能使用页面 URL。

**Q：忘记管理员密码怎么办？**
A：联系开发人员，在 Netlify 环境变量中修改 `ADMIN_PASSWORD` 的值。

**Q：文章删除后能恢复吗？**
A：目前**不支持**从后台恢复已删除文章。删除前请确认，如有重要内容建议先改为「草稿」状态。

**Q：如何让文章出现在 Google 搜索结果中？**
A：① 确保文章状态为「已发布」；② GEO 得分 ≥ 80；③ 前往 [Google Search Console](https://search.google.com/search-console) 提交 Sitemap（地址：`https://skyw.group/sitemap.xml`）；④ 通常 3–7 天后被 Google 抓取。

**Q：ChatGPT / Perplexity 问到公司信息不准确怎么办？**
A：前往 `/admin/seo` → 填写「公司 AI 简介」→ 添加 FAQ → 保存。这些信息会通过结构化数据和 llms.txt 被 AI 学习，通常 2–4 周生效。

**Q：后台登录提示「密码错误」？**
A：请确认使用正确的管理员密码（区分大小写）。如多次失败，联系开发人员确认密码是否已变更。
